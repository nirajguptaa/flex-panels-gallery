# 🌟 Flex Panels Gallery

An interactive flexbox-based gallery where panels expand on click and reveal smooth animated text transitions.  
This project is part of the **[JavaScript30](https://javascript30.com/)** course by [Wes Bos](https://wesbos.com/).  

---

## 🚀 Features
- Responsive **flexbox panels**
- Smooth **expand & collapse animations**
- **Text sliding effects** on panel opening
- **Custom background images** for each panel
- Minimal, clean design using **pure HTML, CSS, and JavaScript**

---

## 🛠️ Tech Stack
- **HTML**
- **CSS (Flexbox, Transitions)**
- **JavaScript (ES6)**

---

## 📖 How It Works
1. Click on any panel to **expand** it.
2. Expanding triggers CSS transitions for:
   - Increasing panel size (`flex-grow`)
   - Sliding in text elements
3. Click again to **collapse** the panel.
